Overview

Midlothian Code is a local Virginia nonprofit organization that reaches out to those learning a programming language or seeking mentoring and networking. Midlothian Code emphasizes free, accessible solutions, such as the online learning paths Free Code Camp and The Odin Project, among others, and seeks to make more guided options available for those who seek them.

After encountering frustration over lack of functionality in existing web services, they chose me to develop a new digital home that could offer the features they needed. Based on an understanding of Midlothian Code's organization type and goals, I recommended implementing a responsive solution using a content management system for ease of updating events, information, and fundraising products.

I partnered with Midlothian Code to design and develop a responsive website that met the needs of its base.

Context and Challenge

At this point, Midlothian Code was a fledgling organization that had not yet registered as a nonprofit and utilized Meetup and Mobilize.io as its primary communication mediums. While Midlothian Code needed more control over its content and web presence, it lacked the budget of a more established nonprofit. In addition, as it needed to have the site launched prior to the messy autumn/winter holiday season, Midlothian Code had a potential timeline of two to four weeks.
